
# Trisha's Portfolio

A modern, responsive personal portfolio website built with **React**, **Vite**, and **Tailwind CSS** — showcasing my skills, projects, experience, and more.

## 🌐 Live Demo

🔗 [https://trisha256.github.io/My_portfolio/](https://trisha256.github.io/My_portfolio/)

---

## ✨ Features

- **Home** — Hero section with introduction, skill pills, and social links
- **About** — Bio, stats (years of experience, projects, clients), and tech highlights
- **Projects** — Showcase of completed projects
- **Skills** — Overview of technical skills and proficiencies
- **Experience** — Professional and academic experience timeline
- **Contact** — Get-in-touch form and contact details
- Fully **responsive** design optimised for all screen sizes
- Smooth **page routing** via React Router

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm

### Installation

```bash
# Clone the repository
git clone https://github.com/Trisha256/My_portfolio.git

# Navigate into the project directory
cd My_portfolio

# Install dependencies
npm install

# Start the development server
npm run dev
```

The app will be available at `http://localhost:5173`.

### Build for Production

```bash
npm run build
```
